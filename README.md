# Inventory Backend

The product inventory backend code

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`TWILIO_ACCOUNT_SID`
`TWILIO_AUTH_TOKEN`
`TWILIO_APP_SERVICE_ID`
`JWT_SECRET`
